API Documentation

Welcome to the API documentation for our President website.

Base URL

The base URL for all API requests is:

http://localhost:5016

Endpoints: Staff

1. Create a new Staff Member

Create a new staff member.

Endpoint: POST /staff

Request:

{
  "id": int,
  "name": string,
  "role": string,
  "email": string,
  "phone": string,
  "imageUrl": string
}

Response (Example):

{
  "id": 1,
  "name": "President One",
  "role": "President of the United States",
  "email": "President@gov.us",
  "phone": "(845) 651-0295",
  "imageUrl": "0b3ce254-8507-4f5b-bda9-afbf5ce60108.jpg"
}

3. Get a Specific Staff Member by ID

Retrieve details of a specific staff member by their ID.

Endpoint: GET /staff/{id}

Response (Example):

{
    "id": 1,
    "name": "President One",
    "role": "President of the United States",
    "email": "President@gov.us",
    "phone": "(845) 651-0295",
    "profilePicture": "0b3ce254-8507-4f5b-bda9-afbf5ce60108.jpg"
}

4. Get a Specific Staff Member by Name

Retrieve details of a specific staff member by their name.

Endpoint: GET /staff/name/{name}

Response (Example):

{
    "id": 1,
    "name": "President One",
    "role": "President of the United States",
    "email": "President@gov.us",
    "phone": "(845) 651-0295",
    "profilePicture": "0b3ce254-8507-4f5b-bda9-afbf5ce60108.jpg"
}

5. Update Specific Staff Member

Update details of a specific staff member by their ID.

Endpoint: PUT /staff/{id}

Request:

{
    "name": string,
    "role": string,
    "email": string,
    "phone": string,
    "imageUrl": string
}

Response (Example):

{
    "id": 1,
    "name": "President One",
    "role": "President of the United States",
    "email": "President@gov.us",
    "phone": "(845) 651-0295",
    "imageUrl": "0b3ce254-8507-4f5b-bda9-afbf5ce60108.jpg"
}

6. Delete a Specific Staff Member

Delete a specific staff member by their ID.

Endpoint: DELETE /staff/{id}

Response (Example):

        204 on success
        4xx-5xx on error

Errors

The API returns appropriate error codes for invalid requests:

Status Code Description
200 OK - The request was successful.
400 Bad Request - The request was invalid or missing parameters.
404 Not Found - The requested resource was not found.
500 Internal Server Error - An error occurred on the server.
